# Data Engineering

## What is it ?
It is basically building data pipelines, processing massive datasets, or deploying machine learning models.

In the real world, Data Engineering follows a concept called ETL (Extract, Transform, Load). It is the backbone of everything a data engineer does.

Here is what it means in plain English:

•	Extract: Grab the raw, messy data from its source (like a website, a file, or a database).
•	Transform: Clean it up, fix the errors, and format it nicely (this is where your Python logic shines).
•	Load: Save that clean data into a final database so the company can use it.
